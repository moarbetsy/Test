Authentication

All API requests require authentication using an API key. Include your API key in the request header:

Authorization: Bearer YOUR_API_KEY
GET

GET

/api/v1/players

Get historical player count data for a specific platform within a date range. Supports filtering by platform and time period.

Request Parameters

platform string (required) - Platform name: "stake" or "shuffle"
start_date string (optional) - ISO 8601 format (e.g., "2025-12-15T00:00:00Z")
end_date string (optional) - ISO 8601 format (e.g., "2025-12-15T23:59:59Z")

Request Example

GET https://api.wagercharts.com/v1/players?platform=stake&start_date=2025-12-15T00:00:00Z&end_date=2025-12-15T23:59:59Z
Authorization: Bearer YOUR_API_KEY

Response

{
"status": "success",
"data": {
"platform": "stake",
"records": [
{
"timestamp": "2025-12-15T03:00:00Z",
"player_count": 66900
},
{
"timestamp": "2025-12-15T03:01:00Z",
"player_count": 67197
}
]
}
}
